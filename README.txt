För att installera angular på din maskin behöver du Node.js https://nodejs.org/en/download/

Om du redan har node.js installera du kan verifiera version med kommando:

 * node -v

Installera Angular CLI 

 * npm install -g @angular/cli@latest

För att skapa en arbetsplat måste måste du ange följande kommando

- ng new app-namn

Så här kör du applikationen 

 - cd my-app
 - ng server --open
